IntelliJ IDEA 단축키(윈도우)

IntelliJ IDEA는 코드 작성, 편집, 검색, 리팩토링, 디버깅 등 IDE 내에서 수행하는 대부분의 작업 명령의 키보드 단축키를 제공한다. 이러한 단축키를 기억해두면 작업 흐름을 원할하게 유지하는데 도움이 된다.

다양한 단축키 중에 많이 사용되는 대표적인 단축키를 알아보자.


IntelliJ 단축키

Alt + Enter
상황에 따른 액션 추천
: 상황에 맞추어 삭제, 코드 최적화, 오류 찾기 등 액션을 추천해준다.

Alt + 1
프로젝트 창 열기 / 닫기
: 프로젝트 창을 열었다, 닫았다하며 화면 전환할 수 있다. 프로젝트 창이 열리면 포커스가 프로젝트 창으로 이동한다.

Alt + F7
함수 사용처 찾기
: 어떤 항목이 사용된 위치를 검색하고 싶을 때 이 단축키를 누르면 어떤 항목이 사용된 모든 곳이 표시된다.

Esc
에디터 창 포커스
: 코드 쓰는 영역인 에디터 창으로 돌아간다.

F2
다음 오류, 경고 제안으로 이동
: 에러, 오류, 경고, 제안이 발생하는 코드 라인으로 점프를 한다. 긴 코드에서 문제 코드라인 잡을 때 유용하다.

Space
프로젝트 창 미리보기
: 프로젝트 창에서 어떤 파일 위 포커스를 위치한 상태에서 이 단축키를 누르면 에디터 창을 열기 전에 미리코드를 볼 수 있다.

Home / End
라인 시작 / 끝 이동

PgUp / PgDn
페이지 위 / 아래 이동

Ctrl + → / ←
단어별 이동

Ctrl + E
최근 실행 파일 열기
: 최근에 작업했던 파일들을 변경순으로 정렬하여 모달 창으로 보여준다.

Ctrl + Shift + E
최근 작업한 코드 위치 검색
: 최근에 작업했던 영역 정보를 목록으로 보여준다. Ctrl + E 와 다르게 파일 안 위치까지 보여준다.

Ctrl + B
선언으로 이동
: 소스코드 상에서 변수에 커서를 두고 이 단축키를 누르면 해당 변수가 선언된 곳으로 이동한다.
소스코드 상에서 클래스 이름 위에 커서를 두고 이 단축키를 누르면 해당 클래스의 파일로 이동한다.

Ctrl Ctrl
실행 목록 창
: 어디에서든 이 단축키를 누르면 어떤 항목이든 실행할 수 있다. Run Anything 창이 열리고 해당 화면에서 실행할 기본 목록이 표시된다. 항목의 이름을 입력하여 다른 실행 구성을 검색하고 실행할 수도 있다.

Ctrl + W / Shift + Ctrl + W
선택 영역 확장 / 축소
: 커서 근처의 코드 선택 영역을 확대하거나 축소할 수 있다. 축소할 시 다시 커서 위치로 축소된다.

Ctrl + Alt + L
코드 컨벤션 정리
: 작성한 코드를 정렬해준다. (들여쓰기 등)
해당 프로젝트의 표준에 맞도록 현재 파일의 서식을 지정할 수 있다.

Ctrl + Alt + Insert
새 파일 생성(에디터 창)
: 이 단축키로 에디터 창에서 새 파일을 생성할 수 있다.

Ctrl + /
줄 주석( //주석 )

: 커서가 위치한 해당 줄을 주석 처리해준다. 이미 주석 처리된 경우 단축키를 누르면 주석 처리가 제거된다.

Ctrl + Shift + /
블록 주석( /*주석*/ )
: 선택된 영역 전체를 주석 처리해준다. 이미 주석 처리된 경우 단축키를 누르면 주석 처리가 제거된다.

Ctrl + Shift + F12
에디터 창 키우기
: 여러 창이 열러있는 상태에서 이 단축키를 누르면 다른 창을 다 닫고 에디터 창만 나타낸다.

Shift + F10
실행

Shift + F9
디버그

Shift Shift
전체 범위 검색
: Shift를 두 번 연속으로 누르면 IDE의 모든 걸 검색할 수 있는 검색창이 표시된다. class 파일, method, 설정 옵션, 액션 등을 검색할 수 있다.

Shift + Ctrl + A
모든 액션 검색
: 모든 단축키에 해당하는 액션들을 검색할 수 있는 검색창이 표시된다.

Shift + Ctrl + Alt + T
리펙토링 요소 활용
: 심벌 또는 영역 위에서 단축키를 누르면 사용가능한 리팩토링 옵션들이 표시된다.

Shift + Ctrl + Enter
현재 구문 완성
: 가장 유용한 단축키이다. 현재 구문에서 미완성되는 부분을 자동으로 완성시켜준다. 대부분의 경우 단순히 코드 끝에 세미콜론을 추가하는 것이다. for문, if문의 경우에는 괄호 및 중괄호를 추가하고 블록 안에 커서를 배치한다. 코드를 추가할 필요가 없을 경우에도 이 단축키를 사용해 적당한 다음 위치에 커서를 배치할 수 있다.
